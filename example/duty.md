## 技术总工程师

**广西路盛通智能科技有限公司 · 研发部**
📅 2020-07 ~ 2024-12

### 核心职责

#### 技术架构与规划
- 🏗️ 负责公司技术架构设计与技术选型，制定前端、3D 可视化、GIS 等技术栈标准
- 📋 主导项目技术方案评审，确保技术方案的可行性、可维护性和可扩展性
- 🔧 建立前端工程化体系，推动代码规范、组件库建设和自动化流程落地

#### 团队管理与培养
- 👥 带领 5-8 人研发团队，负责任务分配、进度把控和技术指导
- 📚 组织技术分享与培训，提升团队整体技术水平
- 🎯 参与团队成员绩效考核，制定个人成长计划

#### 项目交付与质量保障
- 🚀 主导多个大型 3D 可视化项目和 WebGIS 项目的技术实现
- 🔍 负责项目代码审查，确保代码质量和安全性
- 🛠️ 解决项目中的技术难题，优化系统性能和用户体验

#### 技术创新与研发
- 💡 探索 Unity3D WebGL、Cesium、Three.js 等前沿技术在业务中的应用
- 🔬 推动 AI 工具（LangChain、Stable Diffusion）在内部流程中的落地
- 📊 建立技术文档体系，沉淀技术资产

## 前端工程师

**广西中联博远信息技术有限公司 · 软件部**
📅 2025-01 ~ 至今

### 核心职责

#### 前端开发与实现
- 💻 负责公司核心产品的前端开发，使用 Vue 3 + TypeScript + Vite 技术栈
- 🎨 与 UI 设计师协作，实现高质量的用户界面和交互体验
- 📱 开发响应式 Web 应用和跨平台应用（UniApp、Electron）

#### 组件化与工程化
- 🧩 设计和开发可复用的业务组件库，提升开发效率
- ⚙️ 搭建前端工程化体系，包括构建优化、代码规范、自动化测试
- 📦 维护和优化项目依赖，确保项目稳定性和安全性

#### 性能优化与问题排查
- ⚡ 优化页面加载速度和运行性能，提升用户体验
- 🐛 排查和修复线上问题，保障系统稳定运行
- 📈 使用性能监控工具分析和优化关键指标

#### 技术协作与沟通
- 🤝 与后端工程师协作，定义接口规范和数据结构
- 📝 编写技术文档和开发文档，促进团队知识共享
- 🔄 参与需求评审和技术方案讨论，提供前端视角的建议

#### 新技术探索与应用
- 🌟 关注前端技术发展趋势，评估新技术在项目中的应用价值
- 🎯 推动 WebGL、WebGIS、数据可视化等技术在业务中的落地
- 🤖 探索 AI 辅助开发工具，提升开发效率
